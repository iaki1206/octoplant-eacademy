Easy Asset Integrator

The Easy Asset Integrator allows you to import large numbers of assets into Octoplant at once using a simple CSV file — eliminating the need to create each component manually. This is the fastest way to onboard an entire plant's worth of devices.

⏱ 4:44 min 🎯 Efficiency 📋 5 Steps
Tutorial Video – iOpen e-ACADEMY: Easy Asset Integrator
1

Download the CSV Template

The Easy Asset Integrator uses a standardized CSV template. Always start by downloading the template from Octoplant to ensure correct formatting.

1
In the UserClient, navigate to the Extras tab.
2
Click Easy Asset Integrator. The dialog opens.
3
Click Download template to save the CSV template to your computer.
2

Fill in the CSV Template

Open the template in Microsoft Excel or any spreadsheet application and fill in your asset data.

1
Open the downloaded CSV file in Microsoft Excel.
2
Replace the example data with your actual asset information. Each row represents one asset.
3
Ensure the three mandatory columns are filled for every row:
• Name: The component name (e.g., PLC_Line1_Conveyor)
• Directory: The path in the Project tree (e.g., Plant_A/Line_1)
• Component type: The Octoplant component type (e.g., Simatic S7)
🚫
Mandatory FieldsIf any of the three mandatory columns (Name, Directory, Component type) are missing or incorrect for any row, the import will fail for that asset. Double-check all entries before importing.
4
Save the file in CSV format (not XLSX).
3

Import the CSV File

With the CSV file prepared, import it into Octoplant to automatically create all components.

1
In the Easy Asset Integrator dialog, click Import CSV and select your prepared file.
2
Octoplant processes the file and creates all components in the Project tree according to the specified directories.
3
Review the import results. Successfully created components show an Imported status. Any errors are highlighted for correction.
4

Updating Existing Assets

To update an already-imported asset, you need to modify the CSV and re-import it.

1
Re-open the original CSV file.
2
For assets you want to update, delete the value in the 'Imported' status column. This signals to Octoplant that these are existing entries to be updated, not new ones.
3
Make your changes to the asset data in the CSV.
4
Save and re-import the CSV. Octoplant applies the updates to the existing components.
5

Default Job Configurations

After import, Octoplant automatically assigns default job configurations to the new components.

1
After a successful import, navigate to the Jobs section in the AdminClient.
2
Review the automatically created jobs for the imported components. Default configurations are applied based on the component type.
3
Edit the job configurations as needed (see Chapter 5 for detailed job configuration instructions).
